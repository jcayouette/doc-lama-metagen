SOLUTION 5: Post-Processing Filter (QUICK WIN)
-----------------------------------------------
CHANGES TO: internal/ai/generator.go

In sanitize() function, after line 248, add:

```go
// Remove YaST and other irrelevant terms that slipped through
irrelevantPatterns := map[string]string{
    `\bYaST\s+tools?\b`:                  "system tools",
    `\binstallation process\b`:           "setup process",
    `\bSUSE Linux installation\b`:        "system configuration",
    `\bpackage management tools?\b`:      "management tools",
}

for pattern, replacement := range irrelevantPatterns {
    re := regexp.MustCompile(`(?i)` + pattern)
    desc = re.ReplaceAllString(desc, replacement)
}
```
